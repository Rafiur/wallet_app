# Wallet App Frontend

A React frontend for the Wallet App, built with Vite.

## Setup

1. Install Node.js (version 18 or higher) from https://nodejs.org/

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Open http://localhost:5173 in your browser.

## Features

- CRUD operations for all wallet app resources
- Simple, easy-to-use interface
- Connected to Go API at http://localhost:8000/api/v1

## Notes

- Ensure the backend is running on port 8000.
- JWT authentication is not implemented in the frontend yet; add as needed.
- This is a basic implementation; enhance styling and functionality as required.
